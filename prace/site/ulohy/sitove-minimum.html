Síťové minimum

Dokážeš zjistit, co je DHCP, IP adresa a paket?

Lehká úloha

Než se pustíš do složitějších témat, je důležité porozumět několika základním pojmům, se kterými se v sítích setkáš velmi často. V této části si stručně vysvětlíme klíčové termíny, které tvoří základ síťové komunikace a dnes se bez nich neobejdeš.

Zaměř se především na to, k čemu jednotlivé pojmy slouží a jak spolu souvisí. Tyto znalosti se ti budou hodit i v následujících úlohách.

Počítačová/lokální síť

Počítačová/lokální síť je skupina zařízení (například počítače, telefony nebo servery), která jsou mezi sebou propojena tak, aby si mohla vzájemně předávat data a sdílet zdroje. Díky síti mohou zařízení komunikovat například v rámci domácnosti, školy nebo internetu. Zároveň mohou být připojeny mnoha způsoby ať už kabelem nebo bezdrátově.

Síťová karta

Síťová karta (NIC – Network Interface Card) je fyzická součást zařízení, která umožňuje jeho připojení do počítačové sítě. Každé zařízení, které chce v síti komunikovat, musí mít síťovou kartu — ať už kabelovou nebo bezdrátovou.

IP adresa

IP adresa je jedinečný identifikátor zařízení v síti. Funguje podobně jako poštovní adresa — podle ní síť pozná, kam mají být data doručena. Každé zařízení připojené k síti musí mít svou unikátní IP adresu, aby mohlo správně komunikovat s ostatními. Tuto adresu buď nastavujeme ručně (staticky) nebo automaticky (dynamicky) pomocí DHCP serveru.

MAC adresa

MAC adresa je jedinečný fyzický identifikátor síťového zařízení. Je pevně přiřazena síťové kartě výrobcem a používá se především pro komunikaci v rámci lokální sítě. Na rozdíl od IP adresy se běžně nemění.

Výchozí brána

Výchozí brána je zařízení v síti, které zajišťuje komunikaci s jinými sítěmi, nejčastěji s internetem. Pokud zařízení potřebuje odeslat data mimo svou lokální síť, odešle je právě na výchozí bránu (obvykle router), která je předá dál správným směrem.

Maska sítě

Maska sítě určuje, která část IP adresy označuje síť a která konkrétní zařízení. Díky masce dokáže zařízení poznat, zda se cílová IP adresa nachází ve stejné lokální síti, nebo zda je potřeba komunikaci odeslat přes bránu do jiné sítě. Maska se zapisuje buď ve tvaru čísel (např. 255.255.255.0), nebo pomocí prefixu (např. /24).

Paket

Data se po síti nepřenášejí najednou, ale jsou rozdělena na menší části, kterým se říká pakety. Každý paket se skládá ze dvou hlavních částí — hlavičky a datové části. V hlavičce jsou uloženy řídicí informace, například zdrojová a cílová IP adresa, pořadí paketu nebo další údaje potřebné pro správné doručení. Datová část pak obsahuje samotná přenášená data.
Na cílovém zařízení se jednotlivé pakety znovu seřadí a složí se do původní podoby přenášené informace. Jelikož mohou dorazit i v jiném pořadí, než byly odeslány.

Server

Server je zařízení nebo počítač, který je obvykle navržen tak, aby byl dostupný nepřetržitě a mohl poskytovat služby ostatním zařízením v síti kdykoliv jsou potřeba. Může například ukládat soubory, provozovat webové stránky nebo zajišťovat přihlašování uživatelů. Ostatní zařízení v síti (klienti) se k serveru připojují a využívají jeho služby.

Switch

Switch (přepínač) je síťové zařízení, které umožňuje propojit více zařízení v rámci jedné lokální sítě pomocí jednoho centrálního prvku. Přijatá data dokáže na základě MAC adres odeslat pouze tomu zařízení, kterému jsou určena. Díky tomu je komunikace efektivnější než u hubu.

Hub

Hub je jednodušší zařízení pro propojení zařízení v síti. Na rozdíl od switche neposílá data cíleně, ale rozesílá je všem připojeným zařízením. Z tohoto důvodu je méně efektivní a dnes se používá jen zřídka.

Ethernetový kabel

Ethernetový kabel je fyzické médium používané pro kabelové připojení zařízení do sítě. Umožňuje spolehlivý a rychlý přenos dat mezi zařízeními, například mezi počítačem a routerem nebo switchem.

DNS

DNS (Domain Name System) je služba, která převádí doménová jména na IP adresy. Uživatelé si tak nemusí pamatovat číselné IP adresy serverů a mohou používat snadno zapamatovatelné názvy, například www.seznam.cz bude stejné jako 77.75.76.3

Když zadáš do prohlížeče doménové jméno, zařízení odešle dotaz na DNS server, který vyhledá odpovídající IP adresu a vrátí ji zpět. Teprve poté může zařízení navázat spojení se správným serverem.

DHCP

DHCP (Dynamic Host Configuration Protocol) je služba, která automaticky přiděluje zařízením v síti IP adresu a další potřebná nastavení (např. masku sítě, výchozí bránu, DNS servery). Díky DHCP nemusí uživatel nastavovat síť ručně — zařízení získá potřebné údaje automaticky při připojení k síti.

Router

Router je síťové zařízení, které propojuje různé sítě mezi sebou. Nejčastěji spojuje domácí nebo školní síť s internetem a zajišťuje, aby data dorazila do správné sítě. Router také často obsahuje funkce pro zabezpečení a správu síťového provozu.

GUI

GUI (Graphical User Interface) je grafické uživatelské rozhraní — ovládání programu nebo zařízení pomocí oken, ikon, tlačítek a menu místo psaní příkazů v textu. Typickým příkladem je desktop operačního systému nebo nastavení routeru v prohlížeči.
